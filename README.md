# 小红书达人报告生成器

一个可安装的 Codex Skill。输入 WEIQ 达人链接、小红书主页/笔记链接，或直接输入博主名称，即可利用浏览器中已经登录的 WEIQ 与小红书会话生成本地达人报告网页。

报告包含：

- WEIQ 最新图文、视频报价与预估阅读单价
- 达人资料与核心账号指标
- WEIQ 当前一次展示的全部笔记（通常为 16 篇）
- 远程封面、阅读/点赞/收藏数据
- 完整发布文案、可访问的原帖链接
- 汇总指标与适合浏览、分享的单页 HTML

## 安装

在 Codex 中提供本仓库链接，并让 Codex 安装这个 Skill。安装后可直接说：

```text
使用 $build-weiq-influencer-report 分析 迪仔Dizzz
```

也可以把 WEIQ 链接、小红书主页链接、笔记链接或分享短链放在提示词中。

## 使用前提

- 在 Codex 可控制的浏览器中登录 WEIQ 和小红书。
- WEIQ 负责报价与商业数据，小红书负责完整文案和可访问的原帖链接。
- 遇到同名博主时，Skill 会要求确认，不会自动猜测。

## 隐私与数据

Skill 不会把 Cookie、登录凭据或浏览器存储写入报告。生成的达人报告默认保存在本地，也不会提交到本仓库。

## 仓库结构

- `SKILL.md`：完整工作流
- `references/`：输入路由与数据提取规则
- `scripts/`：输入识别和 HTML 报告生成器
- `assets/`：报告网页模板
- `agents/openai.yaml`：Codex 展示信息
